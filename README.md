# schcol
 Flutter Project for Learning
 Name is just for fun 😁

> A simple Food Recipes App 📱
**Used API from https://dummyjson.com/**

**Basice Features:**
 - Authorization 🔏
 - Recipes Listing 📃
 - Recipes filtering ☑️
 - Recipes details 📟

**Flutter Package:**
 - Dio (https://pub.dev/packages/dio)
 - Status Alert (https://pub.dev/packages/status_alert)
